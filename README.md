# STAMPIFY 사이즈 찾기

고객이 원하는 도장 사이즈(세로×가로 또는 지름)를 입력하면 자동으로 적합한 모델을 추천해주는 웹사이트입니다.

## 구성

- `index.html` — 고객용 메인 페이지
- `admin.html` — 관리자 페이지 (구매 링크·썸네일 관리)
- `data.js` — 모델 데이터 & 검색 로직 (두 페이지가 공유)
- `logo.jpg` — STAMPIFY 로고

## GitHub Pages 배포 방법

1. GitHub에 저장소를 만듭니다 (예: `stampify-size-finder`).
2. 이 폴더의 모든 파일을 저장소에 업로드합니다.
3. 저장소의 **Settings → Pages** 에서 `main` 브랜치, `/ (root)` 선택 후 저장.
4. 몇 분 뒤 `https://<username>.github.io/<repo>/` 에서 접속 가능.

관리자 페이지는 `https://<username>.github.io/<repo>/admin.html` 로 접속.

## 검색 로직

- **사각형**: 입력한 세로 이상 & 가로 이상 조건을 모두 만족하는 모델 중 가장 작은 것을 반환.
  - 예: 29mm × 89mm → 38mm × 98mm 규격의 `43103`
- **원형**: 입력한 지름 이상 중 가장 작은 것을 반환.

## 관리자 페이지 사용법

- 각 모델 카드에 **구매 링크** URL과 **썸네일** URL을 입력하면 자동 저장됩니다.
- 썸네일은 외부 URL 입력 또는 "이미지 업로드" 버튼으로 파일 선택 가능 (500KB 이하 권장).
- **내보내기** 버튼으로 설정을 JSON 파일로 저장할 수 있고, 다른 기기에서 **불러오기** 로 그대로 복원할 수 있습니다.

### 주의사항

설정은 사용 중인 **브라우저의 로컬 저장소(localStorage)** 에 저장됩니다.
- 다른 기기/브라우저에서도 같은 설정을 사용하려면 반드시 JSON 내보내기·불러오기를 사용하세요.
- 브라우저 데이터를 삭제하면 설정도 함께 사라집니다.
- 서버 기반 관리가 필요해지면 Google Sheets + Apps Script (기존 ERP와 동일한 방식) 으로 확장 가능합니다.

## 모델 추가/수정

`data.js` 파일의 `RECT_MODELS` 또는 `CIRCLE_MODELS` 배열을 편집하면 됩니다.
